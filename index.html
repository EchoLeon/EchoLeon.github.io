Checksum Compare

Personal project to expand my knowledge with WPF and .NET 9. Checksum Compare is a desktop application that compares the SHA256 hash of two files. It features a little emoji that allows the user to see at a glance whether the hashes are different.

East Surrey College Gym App

College assignment during my apprenticeship to build an application from scratch from specification provided by the college with assistance from my mentor to write code on a professional level.

Python Gym App

A redesign of a college assignment using Python 3 instead. This was a personal project aimed at learning Python and JetBrains' PyCharm IDE. It also utilizes tkinter to draw the form and unittest as the unit testing framework.

In-Game Currency Exchange System (Granblue Fantasy: Relink Mod)

Developed a custom mod that introduces a balanced currency exchange system, allowing players to convert one resource to the other.
